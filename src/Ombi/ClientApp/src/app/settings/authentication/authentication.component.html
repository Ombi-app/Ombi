Authentication

Configure how users authenticate with Ombi, including passwordless login, OAuth, and header-based SSO.

Login methods

Control which authentication methods are available to users.

Allow passwordless login

Users can log in without entering a password. Useful for local or trusted networks.

Enable Plex OAuth

Allow users to authenticate using their Plex account credentials.

Header authentication (SSO)

Authenticate users via a header variable set by a reverse proxy.

Enable header authentication

Use a reverse proxy header to authenticate users automatically.

Enabling header authentication will allow anyone to bypass authentication unless you are using a properly configured reverse proxy. Use with caution!
Header variable name

The HTTP header that contains the authenticated username from your reverse proxy.

Auto-create users from SSO

Automatically create a new Ombi account when an unknown user authenticates via header auth.

New users will be created with the default permissions from your User Management settings.
